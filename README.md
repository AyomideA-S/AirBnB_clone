# AirBnB_clone
An AirBnB clone for the ALX Software Engineering course.

## How to start it
```bash
$ ./console.py
```

## How to use it
```bash
(hbnb) help

Documented commands (type help <topic>):
========================================
EOF  help  quit
```

## Examples
```bash
$ echo "help" | ./console.py
(hbnb)
```
